# 合约交易心得

> 前言
> 
> 珍爱生命,远离合约,我十分反对大家玩合约
> 
> 我因为21年的时候,给客户做项目接触过这个东西,并且必须要熟悉这个逻辑,才走入了约合这条路
> 
> 我再次反对大家玩合约,以下纯属的我个人心得

合约是一种赌博行为,长期游走在合约圈子里的人,10个人有9个人亏损.但是即便是在酷暑难耐的沙漠,缺氧寒冷的雪山,一样也有生物在那里生存,

通过我的多年接触,币圈的确存在一些造富神话,但是,那些神话可能属于庄家,也可能属于鲸鱼,但绝对不属于散户.

如果想赚大钱,想改变人生,请远离.但是我这样的小虾米,我只想在这极端的环境勉强生存,所以我给自己定的目标是平均每日40U,这个收入勉强能够养家糊口,作为副业也很不错.

`所以我在这里记录下我的经验心得`

## 什么样的人适合做合约?
我认为没有任何人适合做合约,就像零下40度环境里生存的牦牛一样,他们也不适合在那个环境生存,只是迫不得已而已.

假设你是一个每月6k收入的打工者,如果你做不到,把6k放在共享单车上,然后安心的回家睡大觉这样的心态,

那么你只会成为币圈的燃料



## 如何做合约?
### **1.每次最多选择三个合约**

选择太多的合约会极大浪费自己的精力,某一个合约爆仓会影响到其他合约.而且久走夜路必闯鬼. `切记:本金第一,亏损第二,收益第三,严格止损止赢`

###  **2.不要盲目的下单**

你可以靠你的运气赚9次,第十次将会带走你的所有的收益.因为陷阱实在太多了,也很难分辨 
`切记:本金第一,亏损第二,收益第三,严格止损止赢`


###  **3.下单前一定要分析合约**

不要认为机会稍纵即逝,然后手快就下单.但据我的经历,大部分看似`机会`,实则是陷阱.这是血的教训.我在这个圈子里只求生存,不求生活.这单即便是机会,错过了不做他也罢.合约圈里,不缺陷进和机会.
`切记:本金第一,亏损第二,收益第三,严格止损止赢`



###  **4.如何选择下单仓位**

`空单` coinglass 爆仓数据 -> 清算最大痛点

`多单` coinglass 爆仓数据 -> 清算最大痛点 

山寨币并不适合逢低做多,因为你根本不知道他能跌到多低.

所有我选择榜前10,根据庄家的资金流入情况来做多.

如果最近流入很多,我会设置大一点的止损和止赢,避免被甩下车.

如果流入很少,我会设置小一点的止损和止赢,苍蝇肉也是肉.



###  **5.止损与止赢**

我的目标是平均每日40U,我最大止赢在150U,最大止损在150U.

完全不扛单那是不可能的,但是要根据情况扛单,要根据庄家资金投入来决定是否要扛单.

即便扛单超过150U,也要立即止损

最近半个月超过400万资金流入,则立即放弃.

说明庄家有备而来的.
`切记:本金第一,亏损第二,收益第三,严格止损止赢`

###  **6.轻仓下单**

每次投入的第一次下单,应该在100-200U,然后 200-400逐步加仓,

超过1000U仓位后,必须停止加仓.进入扛单阶段.记得止损和止赢.

等待也是交易的一个过程,所以要珍惜每一次加仓的机会.

杠杆也不能超过10倍.

当你仓位大之后,很有可能被庄家顶点狙杀,除非你特别有实力
`切记:本金第一,亏损第二,收益第三,严格止损止赢`

###  **7.慎重对待榜前三大哥**

不要轻易的空榜前三大哥,他能做到前三,说明做足了充足的准备.
我以前常常空在半山腰,苦不堪言.
我以为涨幅70%已经到顶了,没想到他能涨到100%,当我以为涨幅到100%就是顶了,没想到却是200%.

经常被捉弄.

所以在做的时候,一定要分析庄家的资金流入大小,然后再决定下单位置.
如果要空榜前三大哥,一定要等待最少2天,等热度过后,分析后再根据情况下单.
`切记:本金第一,亏损第二,收益第三,严格止损止赢`


### **8.高资金费的合约不要碰**

高资金费的合约一般都比较妖,涨跌都非常大.非常容易被打止损.

即便未到止损,进入扛单阶段,高额的资金费让你很难赚到钱.
`切记:本金第一,亏损第二,收益第三,严格止损止赢`

### **9.如何应对横盘的合约**

当横盘以后,说明庄家在观望,或是吸筹,或是诱多,或是诱空.

此时应该密切的观察庄家的资金流入流出,如果是空军,当流入过多以后,

如果资金费为正,赶紧减仓到100u,

如果资金费为负,并且超过0.08%,止赢跑路

`切记:本金第一,亏损第二,收益第三,严格止损止赢`
